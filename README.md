# Analysing-Pharma-Sales-Data

This repository contains a Python-based analysis of daily pharmaceutical sales data. The project uses `Pandas` for data manipulation and `Matplotlib` for data visualization to answer key business questions regarding drug sales trends.

Analysis Highlights
- **Category Sales:** Identified total quantities sold for each ATC drug category.
- **Trend Analysis:** Analyzed peak sales periods for specific months and years (2015, 2016, 2017).
- **Top Performers:** Determined the most sold drug category of 2017 and the highest daily average sales.
- **Seasonal Trends:** Investigated seasonal patterns for Respiratory Drugs (R03), identifying peak demand months.

Technologies Used
- **Python 3**
- **Pandas:** For data cleaning and aggregation.
- **Matplotlib:** For creating bar charts and visual trends.

Key Questions Answered
1. What are the total sales quantities for each drug category (ATC code)?
2. Which three drugs had the highest sales in specific historical periods?
3. Which drug was the best-seller in 2017?
4. Which category has the highest average daily sales?
5. Do Respiratory drugs (R03) show seasonal sales spikes?

Results Preview
Based on the analysis, **N02BE** emerged as the category with the highest average daily sales (~29.92 units), and **Respiratory Drugs (R03)** reached their peak sales during December.

## 📂 Dataset
The analysis is performed on `salesdaily.csv`, which includes daily transactional data categorized by ATC codes.
